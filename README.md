# Northlane Competitive Intelligence Dashboard

Interactive dashboard mapping 47 Northlane Partners clients, 141 competitor targets, and 36+ VC/PE fund connections.

## Local Development

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → New Project → Import this repo
3. Vercel auto-detects Vite — click **Deploy**
4. Done. Live in ~60 seconds.
